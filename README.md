# 图书管理系统（C++）
## 项目介绍
用C++实现的基础控制台图书管理系统，适合考研复试编程练习，核心目标是巩固C++类和对象、链表数据结构的应用。

## 核心功能
1.
 图书信息录入（书名、编号、作者、状态）；
2.
 图书查询（按书名/编号查询）；
3.
 图书借阅/归还（修改图书状态，记录借阅人）；
4.
 图书排序（按编号升序排列）；
5.
 异常处理（输入非数字时程序不崩溃）。

## 技术栈
-
 编程语言：C++
-
 数据结构：单链表（存储图书信息）
-
 开发工具：Dev-C++ / Visual Studio 2022

## 运行步骤
1.
 下载项目文件（main.cpp）；
2.
 用Dev-C++打开文件，编译运行；
3.
 按照控制台菜单提示操作（输入数字选择功能）。

## 学习收获
1.
 掌握C++类和对象的封装、继承；
2.
 熟练使用单链表进行数据的增删改查；
3.
 理解“需求→设计→编码→测试”的工程流程。
